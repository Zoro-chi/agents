## Master AI Agentic Engineering - Build Autonomous AI Agents

### About This Repository

This repository is a comprehensive resource for mastering the development and deployment of autonomous AI agents. Over the course of six weeks, you'll explore and build projects using five powerful Agentic AI frameworks:

- **OpenAI Agents SDK**
- **CrewAI**
- **LangGraph**
- **Autogen**
- **MCP**

![Autonomous Agent](assets/autonomy.png)

### What You'll Learn

- How to design and implement AI agents for various use cases
- The strengths and trade-offs of different Agentic AI frameworks
- Techniques for integrating tools, memory systems, and external services
- Best practices for building scalable, distributed AI systems

### Framework Highlights

- **OpenAI Agents SDK**: Simplifies agent creation with built-in guardrails and structured outputs.
- **CrewAI**: Enables role-based agent collaboration with advanced memory systems.
- **LangGraph**: Provides a graph-based approach for designing complex workflows.
- **Autogen**: Supports dynamic agent creation and distributed architectures.
- **MCP**: Acts as a universal protocol for connecting agents to external tools and services.

### Repository Structure

The repository is organized into multiple directories, each focusing on a specific framework or concept. You'll find:

- Framework-specific labs and projects
- Community contributions and examples
- Guides and troubleshooting resources
- Setup instructions for different operating systems

### Getting Started

To begin, follow the setup instructions provided in the `setup` directory. Each framework has its own dedicated section with step-by-step guides to help you get started.

### Important Notes

- This course involves making API calls to various AI models. Be mindful of potential costs and monitor your usage.
- Explore the guides and troubleshooting resources for additional support.

Enjoy your journey into the world of Agentic AI!
